# Farm Origlena

A farming companion app for smallholder farmers. 
Tagline: "Where crops find their perfect start."

## Features
- **Weather Widget:** Displays mock current and 7-day forecast data.
- **Crop Suggestions:** Recommends crops based on mocked soil, location, and season criteria.
- **Crop Roadmap Calculator:** Based on your land area (Acre, Hectare, Bigha), calculate seed requirements, plant counts, fertilizer schedule, expected yield, and estimated harvest.
- **Offline PWA:** Fully installable App functioning without the internet utilizing browser caching.
- **Multilingual Support:** English, Hindi (हिन्दी), and Marathi (मराठी).

## Tech Stack
- React 19
- Vite
- Tailwind CSS
- React Router DOM
- Motion (for nice fluid animations)
- Vite PWA Plugin

## How to Run

1. Make sure you have Node installed (Node 20+ recommended).
2. From the project root, install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Access the app locally (typically at `http://localhost:3000`).
5. Open browser dev tools and switch to mobile view for best testing experience.
   
To install the PWA, look for the install icon in the URL bar (Chrome/Edge) while viewing the app locally or when deployed.
